Dicionário de Dados

👤 Cliente

Armazena informações sobre os clientes.

Campo Tipo Tamanho Obrigatoriedade Chave Descrição
id_cliente int 11 Sim Primary Key Identificador único do cliente
nome_cliente text 100 Sim - Nome completo
cpf_cliente char 12 Sim Único CPF do cliente
telefone_cliente char 12 Não - Telefone para contato

🚗 Veículo

Armazena informações sobre os veículos dos clientes.

Campo Tipo Tamanho Obrigatoriedade Chave Descrição
placa_carro char 8 Sim Primary Key Identificador único do carro
id_cliente int 11 Sim Foreign Key Referência a Cliente (Cliente.id_cliente)

👥 Funcionário

Armazena informações sobre os funcionários.

Campo Tipo Tamanho Obrigatoriedade Chave Descrição
id_funcionario int 11 Sim Primary Key Identificador único do funcionário
nome_funcionario char 50 Sim - Nome completo
telefone_funcionario char 12 Não - Telefone para contato
id_funcoes int 11 Sim Foreign Key Referência a Funções (Funções.id_funcoes)

🏷️ Funções

Define as funções ou cargos dos funcionários.

Campo Tipo Tamanho Obrigatoriedade Chave Descrição
id_funcoes int 11 Sim Primary Key Identificado único da função
nome_funcao char 50 Sim - Nome da Função

🔧 Ordem de Serviço

Registra as ordens de serviço abertas.

Campo Tipo Tamanho Obrigatoriedade Chave Descrição
id_ordem_de_servico int 11 Sim Primary Key Identificador único da O.S.
valor_servico decimal 10 Sim - Valor do Serviço (Reparo, Peças e M.O.)
status_servico text 20 Sim - Status da O.S.
id_funcionario int 11 Sim Foreign Key Referência a Funcionário (Funcionário.id_funcionario)
placa_carro char 8 Sim Foreign Key Referência a Veículo (Veículo.placa_carro)

📋 Triagem

Detalha a avaliação inicial (triagem) feita no veículo.

Campo Tipo Tamanho Obrigatoriedade Chave Descrição
id_triagem int 11 Sim Primary Key Identificador único da triagem
data_triagem date 8 Sim - Data da realização da triagem
problema text 400 Sim - Descrição do problema encontrado
id_funcionario int 11 Sim Foreign Key Referência a Funcionário (Funcionário.id_funcionario)
placa_carro char 8 Sim Foreign Key Referência a Veículo (Veículo.placa_carro)
id_ordem_de_servico int 11 Sim Foreign Key Referência a Ordem de Serviço (Ordem de Serviço.id_ordem_de_servico)
